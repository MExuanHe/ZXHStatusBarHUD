# ZXHStatusBarHUD
简单易用的状态栏指示器小框架  记住是：小 小 小 

##显示成功信息
```objc
   [ZXHStatusBarHUD ZXH_showSuccess:@"加载成功！"];
```
##显示失败信息
```objc
    [ZXHStatusBarHUD ZXH_showError:@"加载失败!"];
```
##显示加载信息
```objc
    [ZXHStatusBarHUD ZXH_showLoading:@"正在加载中..."];
```
##隐藏
```objc
    [ZXHStatusBarHUD ZXH_hide];
```
